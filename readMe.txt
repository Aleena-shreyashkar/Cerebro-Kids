***************PHP-FILES*******************
config.php - configure database
login.php - enables user to login
register.php - register the user
welcome.php - welcome page opens after successfully login.
logout.php - enables user to logout


*************CSS-FILE****************
style.css - Contains all the css codes used in the task


*************FOLDERS******************
images - contains all images used in this task
db details- exported database sql file


*************Steps to import database******************
1. create a new database with name 'loginr' into phpmyadmin
2. import the file loginr.sql on the 'loginr' database
3. run the application
